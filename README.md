# 👤 Face Detection System

## 📌 Description
This project implements a face detection system using computer vision techniques. It detects human faces in real-time using a webcam or from images and highlights them with bounding boxes.

## 🚀 Features
- Real-time face detection
- Works with webcam and images
- Fast and efficient detection
- Beginner-friendly implementation

## 🛠️ Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

## ⚙️ How It Works
The system uses OpenOpenCV’s pre-trained Haar Cascade model to detect faces in video frames or images. Once a face is detected, a bounding box is drawn around it.

## ▶️ How to Run
1. Install required libraries (opencv-python)
2. Run the Python file
3. Webcam will open and detect faces

## 📌 Use Cases
- Security systems
- Attendance systems
- Basic AI/ML learning
